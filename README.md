# GPT Clone AI Chat Application

A ChatGPT-style AI chat application built using React, Tailwind CSS, Express.js, and OpenRouter API.

## Features

- ChatGPT-like UI
- Sidebar navigation
- AI chat responses
- User and AI message bubbles
- Loading state (Thinking...)
- Enter key message send
- Secure backend API integration
- Environment variable support

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- JavaScript

### Backend
- Node.js
- Express.js
- Axios
- OpenRouter API
